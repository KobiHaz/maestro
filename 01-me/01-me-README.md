# 01-me — Identity Context for AI

This folder holds identity files so AI assistants can work in line with Kobi's style, tools, and context.

## Files

| File | Purpose |
|------|---------|
| `01-me-identity` | Personal profile, business (Xsheva), current projects |
| `01-me-writing` | Writing tone, language, and formatting preferences |
| `01-me-tools` | Tech stack, tools, and workflow preferences |
| `01-me-ai-preferences` | How AI should respond, ask, and proceed |

## Usage

Point AI tools to this folder when you need:

- Consistent voice and tone
- Correct tech stack and tool choices
- Business context (Xsheva, services, products)
- Clear expectations for AI behavior (length, confirmation, when to ask)
