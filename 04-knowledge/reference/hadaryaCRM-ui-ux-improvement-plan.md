# UI/UX Improvement Plan — hadaryaCRM

**Status:** Strategic plan. Direction: "Luxury / Refined".

---

## Principles

- **No purple gradients on white** — avoid AI slop
- **Accent:** teal `hsl(168 45% 38%)` or clay `hsl(25 60% 45%)`
- **Typography:** DM Serif Display for headings, Heebo for body

## Phases

1. **Colors + typography** — `--accent-action`, `--text-hero`
2. **Backgrounds** — subtle gradient, grain overlay
3. **Animations** — card-enter, slide-in-right, staggered reveal
4. **UX** — empty states, loading, accessibility

`src/index.css`, `tailwind.config.ts`, `button.tsx`, `StatsCards.tsx`, `Auth.tsx`, `LeadsEmptyState.tsx`
